CSS History

In 1994, while collaborating with Tim Berners-Lee at CERN, H√•kon Wium Lie first proposed CSS, short for Cascading Style Sheets. The web was primarily text-based at the time, and every browser displayed pages differently. Structure was controlled by HTML, but there was no visual design standard, so everything appeared somewhat different everywhere.

The foundation for elements like fonts, colors, and spacing was laid by the 1996 release of CSS1, the first official version. While CSS3 (which really evolved into a collection of smaller "modules") brought animations, gradients, and flexible layouts, CSS2 added additional layout options and media types a few years later.

Today, CSS continues to evolve with new features like flexbox, grid, and custom properties, making it more powerful and consistent across browsers. What started as a simple styling tool has grown into one of the core languages of the modern web.

CSS Rules and Structure

The browser is instructed by CSS rules on which elements to style and how. A selector and a declaration block are the two primary components of every rule. The HTML element is the target of the selector, and the styles enclosed in curly braces are defined by the declaration block. 

p { color: #333; font-size: 16px; line-height: 1.5; }

The selector p in this example applies the styles to every paragraph element. Every declaration, such as color or font-size, is made up of a property and a value, separated by a colon and ending with a semicolon.

The cascade principle of CSS allows styles to be overridden by inline styles or more specific rules. This flexibility lets developers fine-tune the appearance of content while keeping design separate from the structure of HTML.
